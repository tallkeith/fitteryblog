== README

This is a blog app that I put together for Fittery. This app utilizes RefineryCMS, Ruby on Rails, Bootstrap, Disqus, MaliChimp, and ShareThis.

You should have the following gems installed:

gem 'refinerycms', github: 'refinery/refinerycms', branch: '09d7e11a39c1e0a8a16a05135631db9e0830e092'
gem 'refinerycms-blog', git: 'https://github.com/refinery/refinerycms-blog', branch: 'dff7379738dd1318783dd3135a59a480d3f1432f'
gem 'twitter-bootstrap-rails'
gem 'mailchimp-api', '~> 2.0.6'
